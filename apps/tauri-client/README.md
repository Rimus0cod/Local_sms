# Tauri client

Desktop client for Local Messenger built with Tauri 2, React, and Zustand.

Current scope:

- secure chat desktop shell with a three-column layout
- chat list and chat window
- LAN peer discovery panel
- QR and safety-number verification workspace
- browser fallback backend for frontend-only development

Useful commands:

```bash
npm install
npm run dev
npm run tauri:dev
npm run build
```
